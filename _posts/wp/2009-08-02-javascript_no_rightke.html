JavaScript屏蔽鼠标右键还可以在body标签里加上oncontextmenu=   
self.event.returnvalue=false  
或者:   
   
function nocontextmenu(){    
if(document.all) {   
event.cancelBubble=true;   
event.returnvalue=false;    
return false;    
}}   
或者:   
   
   
另外   
  
oncontextmenu="window.event.returnvalue=false" 将彻底屏蔽鼠标右键
